## 项目说明

### vibe-vue

基于 Vue 3 和 Vite 的前端项目。

#### 技术栈
- Vue 3.5.24
- Vite (rolldown-vite 7.2.5)
- JavaScript

#### 开发命令

```bash
# 进入项目目录
cd projects/vibe-vue

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

#### 项目信息
- 开发服务器：http://localhost:5173/
- 项目版本：0.0.0
