# Vue 3 + Vite

This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

## Development Standards

This project follows strict development standards to ensure code quality and consistency. Please refer to the [documentation](./docs/) for detailed guidelines.

### Quick Start

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Check code quality:
```bash
npm run lint
npm run lint:style
npm run format
```

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Check JavaScript/Vue code
- `npm run lint:style` - Check CSS styles
- `npm run format` - Format code with Prettier
