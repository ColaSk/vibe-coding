# 质量自检与信心评级

调研文档输出后必须按本清单完成自检。深度模式跑全部三类清单；快速模式仅跑"完整性"清单。

## 一、自检清单

### 完整性
- [ ] 所有调研问题都已回答
- [ ] 每个维度至少参考了 2-3 个来源（快速模式可放宽至 ≥ 1 个一级来源）
- [ ] 优势和劣势都有覆盖，无明显偏向
- [ ] 已识别并记录知识盲区

### 准确性
- [ ] 来源权威且时效性好（按 [./sources.md](./sources.md) 阈值标注日期）
- [ ] 矛盾信息已标注并说明分歧来源
- [ ] 关键数据已交叉验证
- [ ] 假设和推测已明确标注

### 可操作性
- [ ] 结论可直接指导决策
- [ ] 风险已量化或分级
- [ ] 提供了替代方案
- [ ] 决策依据清晰

## 二、信心评级

对文档中的每个关键结论标注信心等级：

| 等级 | 范围 | 标准 | 标记方式 |
|------|------|------|----------|
| **高** | 90-100% | 多个权威来源一致，广泛采用 | 直接陈述 |
| **中** | 60-89% | 有文档支持，部分采用，存在小分歧 | 标注"据 X 和 Y 报告" |
| **低** | 30-59% | 来源有限，信息冲突 | 标注"信息有限，仅供参考" |
| **推测** | < 30% | 基于推断，无直接来源 | 标注"推测性结论" |

## 三、信心评级与自检联动规则

信心评级不是孤立标注，必须反向触发自检状态。出现以下情况时，对应自检项**自动判定为不通过**，必须修正后重检：

| 情况 | 自检处理 |
|------|----------|
| 文档中存在 ≥ 1 个"低信心"关键结论 | **准确性**自检不通过，必须补充来源或将表述降级（如"可能"、"据 X 报道"） |
| 文档中存在 ≥ 1 个"推测性"关键结论且未明确标注 | **准确性**自检不通过，必须删除或迁入文末"待验证"附录 |
| 关键决策依据信心 < 中（即低或推测） | **可操作性**自检不通过，必须给出风险标注、替代方案或在结论中明确"待验证"前提 |
| 矛盾来源未标注分歧 | **准确性**自检不通过，必须列出对立观点与各自来源 |
| 文档中无任何信心评级标注 | **准确性**自检不通过（深度模式），全部关键结论必须标注 |

## 四、修正流程

1. 跑完三类清单，记录所有未通过项
2. 应用联动规则，标记被自动触发的不通过项
3. 按"准确性 → 可操作性 → 完整性"顺序逐项修正
4. 修正后重新跑一遍清单，直至全部通过
5. 在文档末尾追加一节"自检记录"（可选），记录关键修正点
